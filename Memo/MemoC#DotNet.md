## Le language C\# #

* Language orienté objet, mixte entre **C++** et **JAVA**.
* Langage pour **Microsoft**
* Framework .NET
* fichier sous extension .cs

##### importation de librairie :

##### les propriétés :

```C++
private string myName;
public string Name
   {
       get
       {
          return myName;
       }
       set
       {
          myName = value;
       }
   }```
